# Projeto CodeHealth

Este projeto implementa um sistema de agendamento médico, e estamos configurando o pipeline de CI/CD.

## Estrutura de Branches

- **main**: Produção
- **develop**: Integração
- **feature/teste-ci**: Funcionalidade temporária

## CI/CD

O pipeline é executado automaticamente toda vez que há um push na branch **develop**. Ele executa uma simulação de build e testes.
